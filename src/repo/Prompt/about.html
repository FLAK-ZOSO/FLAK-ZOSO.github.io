Input

Prompt.py reads commands from the shell, like this:

                C:\>  command argument

The code was written with Python 3.10.1, it can be used with Python 3.10+ without bugs

The Prompt.py files uses the := (walrus) operator, so they will raise a SyntaxError if executed with a version older than 3.8

The Prompt.py files uses match statements with case _: syntax, so they will raise a SyntaxError if executed with a version older than 3.10
