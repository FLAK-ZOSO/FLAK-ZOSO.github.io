v1.2.0 Release notes 30/1/2022

Inline comments are now allowed with //, in source files too

Multiline comments are now allowed with /* opening and *\ closing

off and on are now aliases to false and true in setVar command

exceptions.CommandException only prints the first not recognized word instead of the full input

You can find the complete changelog here

The code was written with Python 3.10.2, it can be used with Python 3.10+ without bugs
